Accessibility

Accessibility is essential to web development. The internet is perhaps the most powerful tool available to people with non-standard access needs, or disabilities. As a large generation born into the digital era starts to age, web accessibility needs to be ready. These reasons need to be at the forefront of our considerations, but the fact is that improved accessibility makes the web better for everyone. Accessibility is so important to the future of web development that the World-Wide Web Consortium (W3C) has been implementing the Web Accessibility Initiative (WAI) since 1997. The current status page for the WAI can be found here. The introduction begins with a quote from Tim Berners-Lee that has become almost cliché, but only almost: The power of the Web is in its universality. Access by everyone regardless of disability is an essential aspect. So whether it’s adding attributes to describe images that can’t be seen, or creating accessible PDFs, Accessibility improves everyone’s web experience.
